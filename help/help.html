Help Me

Getting Started

You Need:
For Audio & Video Operations to work properly:
ffmpeg
Download the ffmpeg.exe & insert it into C:\System32 for easy access.
Or put it in a the JVE / JVE File Location.
For Audio & Video Recording:
Start the jri.exe to create the settings.json
The Program will crash immediately.
Set the OBS Path in settings.json to your OBS Location
The Program needs AdvancedFileOutput so change this in OBS Settings.
Open Broadcaster Software
OBS from Steam
Davinci Resolve

Functions

Audio Reextraction

Your Audio has some Problems?
Program closed without saving etc.
Then hit "A" on your Keyboard or go to "Actions>Reextract Audio"
This may take a while so it is recommended to Reextract with "Auto Switch" toggled on!
This takes some time but meanwhile you can do other stuff... Real life you know?

Send2Resolve

Audio Finished? Perfect!
Now start the DavinciResolve Script in Resolve
Then you can send Data [VIDEO , AUDIO] to Resolve, delete or add some tracks etc.
For more see the
Davinci Resolve Documentation
This takes some time but meanwhile you can do other stuff... Bake Bread! Go Outside! What do Humans in their free time?

RerollThumbnail

Before you can automatically reroll your Thumbnails:
First Step in a new preset : Go to "Programs>Thumbnail" and edit your preset until you like it
Save it & goto "Programs>Edit"
Press "D" and thats it You can automate this by switching "Auto Switch" on!

Generate Audio Composition

Has the current Episode no comp file:
The Program does automatically create one for you. If "Auto Comp" is on
This will take a while and the program might be not responding! This is normal!

Automations

Auto Foreward

Runs to the end of the current Lets Play
If "Auto Comp" is on. It creates a comp.mp3 you for each episode
If "Auto Resolve" is on. It sends each episode to Resolve

Keybinds

Keybind Command
B Switch To previos Episode
N Switch To next Episode
STRG+B jump 10 Episodes backward
STRG+N jump 10 Episodes foreward
+ Change Status in Range 0 to 31 upwards
- Change Status in Range 0 to 31 downwards
A Reextract Audio
S Send Data to Resolve
D Reroll Thumbnail
F Toggle Fullscreen
